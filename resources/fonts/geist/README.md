# Geist Fonts

Son of Anton uses the Geist font family by Vercel (SIL Open Font License).

## Required Files

Download from https://github.com/vercel/geist-font/releases:

- `GeistVF.woff2` — Geist variable font (UI text)
- `GeistMonoVF.woff2` — Geist Mono variable font (editor/terminal)

Place both .woff2 files in this directory.
